== Description ==

**Flexible Scroll Top** plugin enable you to add an easily customizable, mobile-friendly scroll to top button that can match with the design of your website. It uses SVG icon with no jQuery dependency. It's sleek and fast! 

Please see the screenshots below for the range of customizations possible.

== Installation ==

1. Login to your WordPress admin dashboard and go to "Plugins > Add New" page.
1. Search for "Flexible Scroll Top".
1. Click "Install" and then "Activate".
1. Go to "Settings > Flexible Scroll Top" and switch to YES on the "Enable?" option and click "Save".
1. Customize the appearance and behaviour as needed.
